Conceptos de HTML y CSS

En esta pagina podra, leer y recordar algunos fundamentos de HTML, CSS y conceptos básicos necesarios para desarrollar una página web.

Que es html?

Recordemos que HTML no es un lenguaje de programación; es un lenguaje de marcado utilizado para estructurar y organizar el contenido de una página web. Funciona mediante el uso de etiquetas que permiten a los navegadores interpretar y renderizar el contenido correctamente. Para poder diseñar o crear una página web, es importante conocer algunos fundamentos básicos, como los que veremos a continuación.

Propiedades

En HTML existen diferentes etiquetas que permiten agregar texto dentro de una página web. Algunas de las más utilizadas son las etiquetas de párrafo y encabezados, como: 

                <p></p>, 
                <h1></h1>, 
                <h2></h2>, 
                <h3></h3>,
                <h4></h4>, 
                <h5></h5>, 
                <h6></h6>

Estas etiquetas nos permiten mostrar texto con diferentes niveles de importancia dentro de la página. Sin embargo, por sí solas pueden verse muy simples o poco atractivas para el usuario. Para mejorar su apariencia, podemos utilizar CSS, que nos permite aplicar diferentes propiedades de estilo al texto, como el tamaño, color, tipo de letra, alineación, entre otras.

  • color: ; (cambia el color del texto)
  • font-size: ; (cambia el tamaño del texto)
  • font-weight: ; (cambia el grosor de la liena)
  • font-family: ; (cambia la fuente)
  • font-style: ; (cambia el estilo, si es cursiva o no)
  • text-decoration: ; (coloca decoracion, como lineas)
  • text-transform: ; (Cambia de letra mayuscula a minuscula o tipo titulo)
  • text-shadow: ; (Le coloca una sombra al texto)
  • line-height: ; (da un espacioentre parafos)
  • letter-spacing: ; (da un espacio entre letras)

Box Model

El Box Model o modelo de caja es un concepto fundamental en el diseño de páginas web con CSS. En HTML, cada elemento o etiqueta se representa como si fuera una caja. Esto significa que cualquier elemento —como un párrafo, un título, una imagen, una lista o un contenedor— ocupa un espacio dentro de la página. Cada una de estas cajas puede contener contenido en su interior, como texto u otros elementos HTML. Además, estas cajas pueden ser modificadas y estilizadas utilizando CSS para controlar su tamaño, posición y apariencia dentro de la página. El modelo de caja está compuesto principalmente por cuatro partes:

  • Content (contenido): es el espacio donde se encuentra el texto, imagen u otro elemento.
  • Padding: es el espacio interno que separa el contenido del borde de la caja.
  • Border: es el borde que rodea la caja.
  • Margin: es el espacio externo que separa una caja de otras cajas.

Comprender cómo funciona el Box Model es muy importante, ya que permite organizar correctamente los elementos en la página y crear diseños más estructurados y visualmente agradables.

Medidas

En CSS existen diferentes unidades de medida que se utilizan para definir el tamaño de los elementos dentro de una página web, como el ancho, alto, márgenes, tamaños de texto, entre otros. Algunas de las unidades más utilizadas son:

  • Pixeles (px)
  • Rem (rem)

Los pixeles (px) son una unidad de medida absoluta. Un píxel representa un pequeño punto de luz en la pantalla, por lo que cuando usamos px estamos definiendo un tamaño fijo para un elemento. Por otro lado, rem (root em) es una unidad de medida relativa que depende del tamaño de la fuente del elemento raíz del documento (generalmente el html). Normalmente, 1rem equivale a 16px, aunque este valor puede cambiar si se modifica el tamaño de la fuente base en el documento. Utilizar unidades como rem es muy útil para crear diseños más flexibles y adaptables, especialmente cuando se trabaja con páginas web responsivas.

Selectores

Los selectores en CSS son los que nos permiten aplicar estilos a elementos específicos de una página web. Es decir, sirven para indicar a qué etiqueta o elemento HTML queremos cambiarle el estilo. Gracias a los selectores podemos modificar propiedades como el color, tamaño, fuente, márgenes, posición, entre muchas otras. Existen diferentes tipos de selectores, por ejemplo:

  • Selector de etiqueta: aplica estilos a todas las etiquetas de un mismo tipo.
    Ejemplo: p { }
  • Selector de clase: aplica estilos a los elementos que tienen una clase específica.
    Ejemplo: .contenedor{ }
  • Selector de id: aplica estilos a un elemento único dentro de la página.
    Ejemplo: #titulo { }

Comprender cómo funcionan los selectores es muy importante, ya que nos permiten controlar de forma precisa qué elementos de la página queremos modificar y cómo queremos que se vean.

Display

Display La propiedad display en CSS se utiliza para definir cómo se comporta un elemento dentro de la página web y cómo se organiza con los demás elementos. Cada elemento HTML tiene un tipo de visualización por defecto, pero con display podemos cambiar ese comportamiento para controlar mejor el diseño de la página. Algunos valores comunes de display son:

  • block: el elemento ocupa todo el ancho disponible y comienza en una nueva línea.
  • inline: el elemento solo ocupa el espacio necesario y se coloca en la misma línea que otros elementos.
  • inline-block: combina características de inline y block, permitiendo ajustar ancho y alto sin ocupar toda la línea.
  • none: oculta el elemento, haciendo que no aparezca en la página.

Comprender la propiedad display es importante porque permite controlar cómo se organizan los elementos dentro del diseño de una página web.

FlexBox

Flexbox (Flexible Box Layout) es un sistema de diseño en CSS que permite organizar y alinear elementos de forma flexible dentro de un contenedor. Para utilizar flexbox, primero debemos convertir un elemento en un contenedor flexible usando: display: flex; Una vez que un contenedor tiene display: flex, todos los elementos que están dentro de él se convierten en elementos flexibles y pueden organizarse fácilmente. Flexbox permite:

  • Alinear elementos horizontal o verticalmente
  • Distribuir el espacio entre los elementos
  • Crear diseños más adaptables y ordenados
  • Algunas propiedades importantes de flexbox son:
  • justify-content: alinea los elementos horizontalmente.
  • align-items: alinea los elementos verticalmente.
  • flex-direction: define si los elementos se organizan en fila o en columna.

Flexbox es muy utilizado en el desarrollo web moderno porque facilita la creación de diseños responsivos y bien organizados
