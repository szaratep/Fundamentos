Conseptos de HTML y CSS

En esta pagina podra, leer y recordar algunos fundamentos de HTML, CSS y conceptos básicos necesarios para desarrollar una página web.

Que es html?

Recordemos que HTML no es un lenguaje de programación; es un lenguaje de marcado utilizado para estructurar y organizar el contenido de una página web. Funciona mediante el uso de etiquetas que permiten a los navegadores interpretar y renderizar el contenido correctamente. Para poder diseñar o crear una página web, es importante conocer algunos fundamentos básicos, como los que veremos a continuación.

Propiedades

En HTML existen diferentes etiquetas que permiten agregar texto dentro de una página web. Algunas de las más utilizadas son las etiquetas de párrafo y encabezados, como: 

                <p></p>, <h1></h1>, <h2></h2>, <h3></h3>,
                <h4></h4>, <h5></h5>, <h6></h6>
Estas etiquetas nos permiten mostrar texto con diferentes niveles de importancia dentro de la página. Sin embargo, por sí solas pueden verse muy simples o poco atractivas para el usuario. Para mejorar su apariencia, podemos utilizar CSS, que nos permite aplicar diferentes propiedades de estilo al texto, como el tamaño, color, tipo de letra, alineación, entre otras.
  • color: ; (cambia el color del texto)
  • font-size: ; (cambia el tamaño del texto)
  • font-weight: ; (cambia el grosor de la liena)
  • font-family: ; (cambia la fuente)
  • font-style: ; (cambia el estilo, si es cursiva o no)
  • text-decoration: ; (coloca decoracion, como lineas)
  • text-transform: ; (Cambia de letra mayuscula a minuscula o tipo titulo)
  • text-shadow: ; (Le coloca una sombra al texto)
  • line-height: ; (da un espacioentre parafos)
  • letter-spacing: ; (da un espacio entre letras)

Box Model

El Box Model o modelo de caja es un concepto fundamental en el diseño de páginas web con CSS. En HTML, cada elemento o etiqueta se representa como si fuera una caja. Esto significa que cualquier elemento —como un párrafo, un título, una imagen, una lista o un contenedor— ocupa un espacio dentro de la página. Cada una de estas cajas puede contener contenido en su interior, como texto u otros elementos HTML. Además, estas cajas pueden ser modificadas y estilizadas utilizando CSS para controlar su tamaño, posición y apariencia dentro de la página. El modelo de caja está compuesto principalmente por cuatro partes:

  • Content (contenido): es el espacio donde se encuentra el texto, imagen u otro elemento.
  • Padding: es el espacio interno que separa el contenido del borde de la caja.
  • Border: es el borde que rodea la caja.
  • Margin: es el espacio externo que separa una caja de otras cajas.
Comprender cómo funciona el Box Model es muy importante, ya que permite organizar correctamente los elementos en la página y crear diseños más estructurados y visualmente agradables.
